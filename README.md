# Fix My Smart Home Now

A No-Dead-Zones-style site focused on smart home reliability, troubleshooting, and practical buying guidance.

## Initial positioning

- Core promise: fix flaky smart home setups without guesswork
- Focus: reliability over novelty
- Angle: clear decision trees, structured hubs, and troubleshooting that leads to the right fix

## Initial topic clusters

- Connectivity and pairing
- Zigbee / Z-Wave / Thread / Matter decisions
- Wi-Fi device limits and router strain
- Smart plugs / lights / switches disconnecting
- Best hubs / bridges for mixed ecosystems
- Reliability-first buying guides

## Operating model

Use the NDZ playbook:
- start page + strong hubs
- problem-solving first, commerce second
- source-map contract
- regression guard
- safe cron-driven maintenance
